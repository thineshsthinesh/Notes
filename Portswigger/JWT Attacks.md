
# What are JWTs ? 

**JSON web Tokens (JWTs)** are standardized format for sending signed JSON data between systems. These data can be part of authentication, session handling, and access control mechanisms. 

JWT stored in the **local storage on the client side** rather than server. Popularly used in **distributed websites** so users can interact with multiple backend servers. 

## JWT Format 

JWT consists of 3 parts : 
- **Header**
- **Payload**
- **Signature**
